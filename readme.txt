Shortest Path Using A* Start Algorithm

by chengcheng Mou and Ran Rui, Feb 2013.

Building Evironment Required:
	Linux, Mac or Windows
	Qt 5.0.1 with Qt Creator 2.6.2
The executable file provided in the package is built in Linux. To run the program, change working directory to the project directory, and type "./AI_Project".

Instructions:
    There is a location file and a connection file in the package. When specifying input file path, type 0 to use these default files, otherwise type the path of your input file.
    The code only allow one node to be excluded from search. Type arbitrary string if you do not want to exclude any node from search.
